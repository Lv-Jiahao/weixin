```bash
5. 对应的Python版本       (1小时)  - 参考实现方法
```

### 第二步：构建项目

```bash
# 方式1: CMake (推荐)
mkdir build && cd build
cmake ..
make -j4

# 方式2: Makefile
make -j4

# 方式3: 脚本
chmod +x scripts/build.sh
./scripts/build.sh
```

### 第三步：运行示例

```bash
# 运行主程序框架
./build/bin/satellite_sim -c config/config.json -s 10000 -v

# 自定义参数
./build/bin/satellite_sim -c config/my_config.json -s 50000
```
